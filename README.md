# Консольное приложение для управления библиотекой книг
## Тестовое задание для "Effective Mobile"

Приложение позволяет пользователю с помощью консольных команд добавлять, удалять, искать и отображать книги, а также менять статус книги.

###  Технологии

Для написания использован Python 3.12. Для пользователя приложения будет наиболее надёжно убедиться в использовании данной версии Python.

###  Данные и хранение

Каждая книга имеет id, название, автора, год издания, статус: “в наличии” или “выдана”. Информация о книгах хранится в текстовом файле, изображающем базу данных.

При записи книги в файл ей присваивается id, начиная с 1, и с тех пор каждый следующий присваиваемый id увеличивается на 1 вне зависимости от того, остаются или удаляются книги с меньшими id из файла.

### Валидация

В приложении работает валидация вводимых данных, например, id должен быть целым числом, год должен состоять из 4 цифр (скажем, что у нас библиотека достаточно современных книг), статус книги может быть только либо “в наличии”, либо “выдана” и т. п. Если пользователь ввёл неверное значение, ему будет предложено повторить ввод либо прервать команду, если он никак не может ввести валидное значение. В случае прерывания команды пользователь снова получит предложение ввести команду и сможет как ввести её повторно, так и выбрать другую команду.

### Основные команды

Есть 5 основных команд:
1) **добавить книгу**: пользователь вводит название, автора и год издания, после чего книга добавляется в библиотеку с уникальным id и статусом “в наличии”;
2) **показать все книги**: пользователю выводится список всех книг с их id, названием, автором, годом издания и статусом;
3) **найти книги**: пользователь может искать книги по названию, автору или году издания;
4) **изменить статус книги**: пользователь вводит id книги и новый статус (“в наличии” или “выдана”);
5) **удалить книгу**: пользователь вводит id книги, которую нужно удалить.
